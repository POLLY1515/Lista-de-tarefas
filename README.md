# 📋 Lista de Tarefas

Aplicação web para gerenciamento de tarefas, desenvolvida com foco em prática de JavaScript, manipulação do DOM e armazenamento local.

---

## 🚀 Demonstração

👉 Acesse o projeto online:  
🔗 https://polly1515.github.io/Lista-de-tarefas/

---

## ⚙️ Funcionalidades

- ✅ Adicionar tarefas
- ⌨️ Adicionar tarefas com a tecla Enter
- ❌ Impedir tarefas vazias
- ✔️ Marcar tarefas como concluídas
- 🗑️ Remover tarefas
- 💾 Persistência de dados com localStorage
- 🔢 Contador de tarefas
- 🔍 Filtro de tarefas:
  - Todas
  - Pendentes
  - Concluídas
- 📱 Layout responsivo

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage

---

## 📚 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Manipulação do DOM
- Eventos em JavaScript
- Organização de código
- Persistência de dados no navegador
- Construção de interfaces interativas
